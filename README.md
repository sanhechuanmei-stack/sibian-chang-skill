# 思辨场 / Sibian Chang

一个给 Claude Code 用的思辨辩论 skill。输入任何命题或问题，自动召集5位来自不同领域的历史/当代人物，从不同维度展开多轮深度辩论，最终生成可直接用于内容创作的选题框架。

---

## 效果预览

输入：`/思辨 年轻人躺平是因为没有上进心`

你会得到：
- **命题解剖**：核心假设、涉及维度、潜在争议点
- **5位人物**：自动选角，覆盖主流/反主流/异类/边缘/实践5个席位
- **风格档案**：每人的用词习惯、思维结构、攻击方式、切入维度各不相同
- **多轮辩论**：每轮有不同推进目标，从亮立场→攻前提→极端案例→结盟→承认一半
- **你可以插话**：直接点名怼某人，全场跟着一起响应
- **内容创作角度**：5个来自不同认知层面的选题框架，每个含标题建议+核心论点+最反直觉的一句话
- **自动保存**：完整记录存入 Obsidian

## 安装

1. 创建目录并下载 skill 文件：

```bash
mkdir -p ~/.claude/skills/sibian-chang
curl -o ~/.claude/skills/sibian-chang/SKILL.md \
  https://raw.githubusercontent.com/sanhechuanmei-stack/sibian-chang-skill/main/SKILL.md
```

2. 重启 Claude Code，skill 自动生效。

---

## 使用方法

```
/思辨 你的命题或问题
```

例如：
```
/思辨 AI 正在夺取人的思考能力
/思辨 努力真的能决定成功吗
/思辨 现代人为什么越来越孤独
```

恢复上一场未完成的思辨：
```
/思辨 继续
```

### 辩论中的指令

| 指令 | 效果 |
|------|------|
| 回车 | 继续下一轮 |
| `i` | 插话，直接点名怼某人或抛出新观点 |
| `s` | 保存进度并暂停 |
| `q` | 结束辩论，进入收尾流程 |

## 保存路径

辩论记录自动保存到 Obsidian，路径：

```
~/Documents/Obsidian Vault/3.Sources(原材料)/思辨场/YYYY-MM-DD-命题.md
```

如果你的 Obsidian 路径不同，打开 `SKILL.md` 修改第9步里的保存路径即可。

## 设计理念

- **5个固定席位**：主流/反主流/异类/边缘/实践，刻意制造张力
- **切入维度强制分配**：从定义层、机制层、心理层、结构层、后果层、反问层、受益者层等10个维度动态选5个，确保每人从完全不同的认知层面进入命题
- **论证链硬规则**：每个论断必须完成"论断→因为→所以→这意味着"的完整链条，不允许只抛结论
- **你是第六个参与者**：不只是旁观，可以直接下场点名开炮

## 适用场景

- 对一个观点或问题做多维度深度思考
- 打破思维定势，寻找反直觉角度
